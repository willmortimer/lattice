Lattice is a fast, local-first desktop workspace where canonical content stays
in a real directory. It currently combines Markdown pages, links and backlinks,
search, basic canvases, SQLite-backed tables, command history, and native file
interoperability inside a Tauri application.

The project is in active development. Phase 0—the headless storage and command
core—is complete, and Phase 1—the fast local notebook and desktop shell—is
underway.
