Lattice is a free, local-first desktop workspace for notes, canvases, structured
data, notebooks, and local tools. A workspace is an ordinary directory, so your
work remains readable in other editors, visible to backup software, and useful
without an account or internet connection.

## The short version

1. Create or open a normal directory as a workspace.
2. Use **⌘/Ctrl+P** to create resources or jump anywhere.
3. Use **⌘/Ctrl+N** for Quick Note and **⌘/Ctrl+K** for search.
4. Open **Inspect** for links, history, schema, source, and diagnostics.
5. Keep using the same files from Finder, Git, VS Code, scripts, and other apps.

## Product and service status

The local app already includes pages, Quick Note, canvas, SQLite data apps,
Parquet/DuckDB analytics, charts, maps, notebooks, tasks, bounded workflows,
proposals, artifacts, interfaces, and local search foundations.

Lattice Cloud is a future optional paid service for sync, sharing, managed
backups, teams, hosted execution, and publishing. It is not required for the
local app. See [current availability](/docs/current-product/) and
[cloud and sync](/docs/cloud-and-sync/) for the honest boundary.
