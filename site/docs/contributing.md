Development happens in the
[Lattice repository](https://github.com/willmortimer/lattice).

Before architectural or product-surface changes, read the root `AGENTS.md` and
the canonical vision, principles, system architecture, frontend performance,
capability-discovery, roadmap, and design-review documents it lists.

Accepted irreversible decisions live in
[`docs/decisions/`](https://github.com/willmortimer/lattice/tree/main/docs/decisions).
New irreversible choices should receive an ADR instead of silently becoming
implementation facts.
