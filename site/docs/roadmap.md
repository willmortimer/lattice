---
title: Roadmap
description: The staged path from the current local notebook to the wider information studio.
---

Phase 0 established the specifications, storage abstractions, semantic command
core, and CLI. Phase 1 is the current focus: the native shell, page editing,
links, search, basic canvas, resource navigation, external-edit reconciliation,
and performance foundations.

Later phases add:

1. Relational data applications and multiple views.
2. Analytical datasets, DuckDB, Parquet, Arrow, charts, and dashboards.
3. Local APIs, MCP, notebooks, Python, and programmable artifacts.
4. Durable automation, remote data, publishing, plugins, sync, collaboration,
   mobile, and native ink.

The ordering is architectural rather than a release-date promise. See the
canonical [roadmap](https://github.com/willmortimer/lattice/blob/main/docs/29-roadmap.md).
