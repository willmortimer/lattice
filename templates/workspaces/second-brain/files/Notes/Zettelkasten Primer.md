# Zettelkasten Primer

One note, one idea. The slip-box model scales because links carry structure that folders cannot.

## Rules of thumb

1. Write for a stranger — including future-you.
2. Link every note to at least one neighbor; see [[Notes/Linking Philosophy]].
3. Add to [[MOCs/Knowledge Base]] when a cluster matures.

## Lattice fit

- Capture in [[Inbox/]] via [[Notes/Capture Workflow]].
- Store atoms in [[Notes/]].
- Cite originals in [[Sources/]].
