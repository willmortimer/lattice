# Ops hub

The operational memory of your team — meetings, one-on-ones, reviews, and goals.

## Meetings

- [[Meetings/]] — one dated page per meeting; today's seeded example: [[Meetings/{{date}} Kickoff]]
- **⌘N** captures straight into `Meetings/` using [[Templates/Daily]]
- New agenda? Start from [[Templates/Meeting Note]]

## People

- [[1-1s/]] — one page per person, newest entry on top, using [[Templates/One-on-One]]

## Cadence

- [[Reviews/]] — end each week with [[Templates/Weekly Review]]
- Track goals in the **OKRs** table (`OKRs.data`) — one row per key result

## Hygiene

- Decisions get their own heading in the meeting page, so search finds them
- Finished cycles move to [[Archive/]]
