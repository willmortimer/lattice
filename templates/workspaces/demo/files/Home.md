# Home

A sample Lattice workspace. Everything here is an ordinary file under a real
directory — open it in any editor, or stay inside Lattice.

## Try this

1. Search with **⌘K** — pages are indexed by path and body.
2. Open `Canvases/Product Strategy.canvas` — double-click a file node to jump.
3. Capture a quick note with **⌘N** into `Inbox/`.
4. Open `CRM.data` for a small typed table (contacts).
5. Browse `Resources/` for JSON, YAML, TypeScript, and an SVG mark.
6. Open [[Research/Architecture]] for a Mermaid diagram in a page.
